Git Basics

Learn how to get started with Git: installation, repositories, commits, and pushing to GitHub.

Installing Git

Download and install Git from the official website: git-scm.com. Follow the instructions for your operating system.

Creating a Repository

Initialize a local repository with:

git init

Or clone an existing repository:

git clone <repository-url>

Committing Changes

Track changes and save snapshots:

  • git add <file> - Stage changes
  • git commit -m "message" - Commit changes

Check status of your files:

git status

Pushing to GitHub

Upload your commits to a remote repository:

git remote add origin <repository-url>
git push -u origin main
